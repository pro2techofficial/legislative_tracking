MUNICIPAL LEGISLATIVE TRACKING SYSTEM
PHP + MySQL + XAMPP + VS Code

INSTALLATION:
1. Copy this folder to:
   C:\xampp\htdocs\mlts

2. Open XAMPP Control Panel.
   Start Apache and MySQL.

3. Open browser:
   http://localhost/phpmyadmin

4. Create a database named:
   mlts_db

5. Import:
   database.sql

6. Open:
   http://localhost/mlts

Username: admin
